# Project Definition

## Project Name
AI Code Assistant

## Purpose
A visual tool that helps developers understand how a full-stack codebase is structured and how different parts connect and flow together.

## Main Goal
To help developers visualize code relationships, endpoint flows, and frontend-to-backend interactions using graphs and AI insights.

## Core Features
- Visual graph of the system
- Explorer for pages, endpoints, controllers, services, and repositories
- Flow tracing for actions and endpoints
- Insight panel with AI explanations
- Endpoint list view
- Subtle futuristic UI

## MVP Focus
- Use mock data first
- Visualize one or two full-stack flows
- Allow node selection and flow highlighting
- Show AI explanations for selected nodes and flows
